# Practica 9 - Text Analysis

## Dataset

Se utilizó el dataset *Coffee Distribution Across 94 Countries*, el cual contiene información sobre producción, consumo, importación y exportación de café en distintos países y años.

El dataset cuenta con:
- 6016 registros
- 21 variables
- Datos de 94 países

## Objetivo

El objetivo de esta práctica es realizar un análisis de texto utilizando la variable de países, con el fin de identificar cuáles aparecen con mayor frecuencia dentro del dataset mediante una visualización tipo word cloud.

## Metodología

Se realizaron los siguientes pasos:

1. Carga del dataset limpio (`coffee_cleaned.csv`).
2. Selección de la columna *country*.
3. Conversión de los datos a texto continuo.
4. Generación de una nube de palabras (word cloud).
5. Visualización de los resultados.

## Resultados

- Se generó una word cloud que muestra los países más frecuentes en el dataset.
- Se observó que algunos países aparecen con mayor tamaño, indicando mayor frecuencia.
- Destacan países como Congo, República Centroafricana, Brasil y Costa Rica.
- Se identificó una mayor presencia de países de ciertas regiones, especialmente África.

## Conclusiones

El análisis de texto permitió identificar los países más representados en el dataset de forma visual e intuitiva.

La word cloud facilita la detección de patrones en la distribución geográfica de los datos.

Este tipo de análisis complementa los métodos cuantitativos y ayuda a obtener una visión general del dataset.
